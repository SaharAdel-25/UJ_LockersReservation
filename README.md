**🔐 نظام حجز الخزائن | UJ Lockers Reservation System**


نظام **UJ Lockers Reservation** لإدارة خزائن الجامعة يتيح للطلاب إنشاء حساب أو تسجيل الدخول، استعراض المباني المتاحة، وحجز اللوكرات. اللوكرات المتاحة تظهر باللون الأخضر والمحجوزة باللون الأحمر، ويمكن للمستخدم حجز لوكر واحد فقط لكل مبنى وبحد أقصى لوكرين في مبنيين مختلفين. بعد اختيار اللوكر، يوفر النظام خيارات الدفع (كاش أو بطاقة) ويعرض فاتورة الحجز.

**UJ Lockers Reservation** is a university locker management system that allows students to create an account or log in, browse available buildings, and reserve lockers. Available lockers appear in green, reserved lockers in red, and users can reserve only one locker per building with a maximum of two lockers in different buildings. After selecting a locker, the system provides payment options (cash or card) and generates a reservation invoice.
